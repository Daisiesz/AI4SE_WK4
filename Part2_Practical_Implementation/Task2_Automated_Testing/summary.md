# Automated Login Test Summary 

This automated test used Selenium WebDriver to simulate a user logging into a sample web page.  
The script automatically entered a username and password, clicked the login button, and verified the successful login through page navigation. 

**Tools Used:** 
- Python 3.13 
- Selenium 4 
- WebDriver Manager (for automatic Chrome driver setup)  

**Result:** 
The automation successfully logged in and captured a screenshot (`login_result.png`), confirming the functionality of automated web interaction for login forms.
